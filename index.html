JQueryLens is your lens component

JQueryLens is the component you'll love for zooming on any image or set of images.
See it working here.

To start using it, you can clone the project from github and switch to the production branch: 

$ cd your_workspace
$ git clone http://github.com/Agil-AZ/JQueryLens
$ git checkout production

How to use

We've crafted this component to be flexible but also easy to use. You can see a basic example of use in the file samples/basic.html, and an advanced one in samples/advanced.html.

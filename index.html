JQueryLens

A JQuery based Lens component

Download Download fork JQueryLens on GitHub

JQueryLens is your lens component

JQueryLens is the component you'll love for zooming on any image or set of images.
See it working here.

To start using it, you can clone the project from github and switch to the production branch: 

$ cd your_workspace
$ git clone http://github.com/Agil-AZ/JQueryLens
$ git checkout production

How to use

We're crafting this component to be flexible but also easy to use. You can see a basic example of use in the file sample/index.html. We hope to craft an advanced one soon.

As you can see in the basic example, you just need to place in your html code a couple of <div> elements: one for the thumbnail of the image and the other for the zoom area: 

<div id="thumbnail" class="resize">
    <img src="images/image.jpg" alt="imagen" />
    <div id="locator">
    </div>
</div>
<div id="lens">
    <img id="view" src="images/image.jpg" alt="imagen" />
</div>
